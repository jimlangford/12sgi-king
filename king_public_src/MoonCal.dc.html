Neo4j · agenda_civic layer · example queries
Meetings on Kū nights (stand-testify) in this cycle:
MATCH (m:CivicMeeting)-[:IN_PO]->(po:MoonNight)
WHERE po.game_frame = 'stand-testify'
RETURN m.date, m.body, m.tenant, po.name ORDER BY m.date

Walk a project backward through prior meetings:
MATCH path = (i:AgendaItem)-[:PREV_ACTION*1..6]->(prev)
RETURN i.title, [n IN nodes(path) | n.date] AS timeline

Items touching money-chain orgs:
MATCH (i:AgendaItem)-[:NAMES]->(o:AgendaOrg)-[:SAME_AS]->(ent)
RETURN i.title, o.name, ent.name ORDER BY i.date

Full moon civic events (Māhealani):
MATCH (m:CivicMeeting)-[:IN_PO]->(po:MoonNight {name:'Māhealani'})
RETURN m.date, m.body, m.url
