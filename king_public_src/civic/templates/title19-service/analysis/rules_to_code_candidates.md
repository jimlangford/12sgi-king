# Rules → Code: Candidates for Codification (AG-first batch)

> **LABEL: ANALYSIS / POSITION — NOT LAW.** This document is editorial analysis identifying provisions that currently live in (or are delegated to) the Maui Planning Department **administrative rules** and that, in our assessment, arguably should be **codified into Title 19** of the Maui County Code. Every underlying citation to the Code is sourced and verbatim-checked; the recommendation itself is opinion offered to inform the ongoing Title 19 rewrite. It is not legal advice and does not state current law.

> **UPDATE 2026-06-17 — rule text now in hand; candidates FIRMED UP (EXPANDING → SOURCED):**
> - **A3 (civil-fine schedule) → SOURCED.** The implementing rule is **Rule §15-2** (Title MC-15, "Admin Procedures & Civil Fines for Violations of Titles 16, 18 & 19," eff. 9/3/1993; Imp: MCC §19.530.030). It carries the full §15-2-9 fine schedule (e.g., development standard $1,000 + $100/day; building w/o permit $200 + $50/day; continuing-violation doubling to a $1,000/day max; repeat doubling) and the §15-2-8 correction-time schedule. **Recommendation firmed:** this citizen-facing penalty schedule (and the 30-day finality + Board-appeal route) should sit in a Code-anchored schedule, not only in a 1993 agency rule. *(Verbatim: corpus/raw/county_docs/civil_fines_rule_mc15_ch2_15-2.md)*
> - **A1 / A2 / B4 (AG rule content) → SOURCED.** AG admin rule **Ch. 102** (eff. 2019) is now ingested: it defines the **Declaration** requirement (§12-102-4) and farm-dwelling / commercial-structure / open-rec procedures (§§12-102-5..-7), appeal to BVA (§12-102-11). The farm-plan/declaration content the Code leans on (§§19.30A.050/.072) is therefore rule-defined → codify minimum contents. *(Verbatim: corpus/raw/county_docs/ag_district_admin_rules_ch102.md)*
> - **Cross-title note:** Rule §15-2 binds the same civil-fine machinery to Titles 16, 18 AND 19 — a strong argument for a single codified, cross-title penalty schedule (ties to C7).

**Method.** We read the Agricultural District chapter (MCC 19.30A) and the Enforcement chapter (MCC 19.530) verbatim from Municode (retrieved 2026-06-17) and identified the **delegation points** where the Code hands substance to administrative rules, plus topics the Code regulates only thinly while a companion administrative rule carries the operative detail. Where a candidate depends on the *content* of an administrative rule whose full text is not yet ingested, it is marked **(rule text = EXPANDING)** and the recommendation is framed conservatively.

---

## A. Express delegations in Title 19 that push substance to rules (highest-confidence candidates)

These are points where the **Code itself** says the detail lives in administrative rules. Codifying the core standard (while leaving genuinely procedural mechanics to rules) increases public notice, democratic accountability (council adoption vs. agency adoption), and resistance to quiet administrative change.

1. **Commercial agricultural structures — substantive regulation via rule.**
   - Code hook: *"Additional regulation of commercial agricultural structures may be established by administrative rules."* — MCC **19.30A.072(J)**.
   - Also delegated: registration, database, exemptions, size/permit thresholds sit in 19.30A.072(A)–(I), but (J) leaves an open-ended grant.
   - **Position:** Any operative *standards* (not mere forms) added by rule under 19.30A.072(J) should be elevated into 19.30A.072 so the permitting public sees them in the Code. *(rule text = EXPANDING)*

2. **General AG rule-making authority.**
   - Code hook: *"The planning director and the director of public works and waste management shall have the authority to adopt rules regarding the administration of this chapter."* — MCC **19.30A.120**.
   - **Position:** Identify which adopted AG rules state binding *use/standard* requirements (e.g., farm-plan content, gross-sales verification mechanics, ag-tourism operational limits) vs. pure procedure; codify the binding requirements. *(rule text = EXPANDING — see Maui Planning administrative rules "MC-12, Dept. of Planning, Subtitle 01 — Rule Amendments for Chapter 19.30A Relating to the Agricultural District")*

3. **Administrative enforcement procedure & civil-fine mechanics.**
   - Code hook: § **19.530.030** authorizes the director to act *"pursuant to this chapter and such administrative rules as the director may adopt."*
   - Companion rule: *"Rules for Administrative Procedures and Civil Fines for Violations of Chapter 16.13 and Title 19."*
   - **Position:** The **fine schedule, default amounts, and mitigation/escalation factors** carried in the civil-fines rule are exactly the kind of citizen-facing penalty detail that belongs in the Code (or in a Code-referenced schedule), not solely in agency rules. *(rule text = EXPANDING)*

---

## B. Code topics regulated thinly while operative detail likely sits in rule/form (medium-confidence)

4. **Farm-plan definition and required contents.** The Code repeatedly conditions rights on "an approved farm plan" (e.g., 19.30A.050(B)(2)(c), (B)(11)(b)(iii), (B)(12); 19.30A.072(A)(2)) but does not define the farm plan's required contents in the chapter. **Position:** codify minimum farm-plan contents and approval criteria; today they are effectively defined by department practice/forms (e.g., the County "Farm Dwelling and Agricultural Structures Declaration"). *(rule/form text = EXPANDING)*

5. **Gross-sales verification ($35,000) mechanics.** Multiple rights hinge on "$35,000 of gross sales … as shown by state general excise tax forms and federal form 1040 Schedule F" (19.30A.050(B)(2)(a), (B)(11)(a)). The threshold is codified; the **verification/audit procedure** is administrative. **Position:** codify the verification standard (what counts, look-back, recordkeeping) to prevent drift. *(rule text = EXPANDING)*

6. **Agricultural tourism registration.** 19.30A.050(B)(15) codifies operating limits (8am–6pm, registration, suspension on 3 violations) but routes the registration **form and process** to the department. **Position:** confirm whether any *eligibility* criteria live only in the registration process; codify if so. *(rule/form text = EXPANDING)*

---

## C. Cross-cutting structural recommendation

7. **Single codified penalty schedule.** Civil-fine caps are in the Code (§ 19.530.030: ≤$1,000; ≤$1,000/day; elevated $20,000 / $10,000-day for unpermitted transient accommodations; ≤1% project cost), but the **graduated default amounts actually assessed** appear to live in the civil-fines administrative rule. **Position:** publish a single Code-anchored penalty schedule so the public can determine exposure from the Code alone.

---

## How to read the confidence tags
- **High (Section A):** the Code text *itself* delegates substance to rules — codification candidacy is established by the Code, independent of reading the rule.
- **Medium (Section B):** candidacy depends on confirming that operative content sits in the rule/form; flagged EXPANDING pending full rule-text ingestion.
