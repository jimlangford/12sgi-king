Hawaiʻi State — who lobbies and pays the deciders

Entities that appear in both public records at once: registered to lobby the State of Hawaiʻi (Ethics Commission) and a campaign donor to Hawaiʻi State candidates (Campaign Spending Commission). Lobbying is lawful; donating is lawful. Doing both is a double channel — two ways the same interest reaches the same decider — so it belongs on the map, as a question.
