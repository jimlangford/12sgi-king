# Tenant Assistant Plan

Overview

The Tenant Assistant helps guide tenants through common workflows using a guided interview, timeline builder, doc generator, and evidence vault. It will be AI-assisted and integrate with the case management engine.

Phases
- Prototype guided interview UI components
- Build timeline/editor and evidence upload flows
- Integrate AI guidance for document drafting and next-step suggestions
