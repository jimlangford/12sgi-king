# Case Management Plan

Overview

All work converges on a case-based data model. A Case contains timeline entries, documents, notes, status, and AI context.

Phases
- Design canonical case schema
- Implement API for CRUD and timeline events
- Attach documents and evidence
- Add permissions model per tenant and staff
