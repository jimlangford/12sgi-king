# govOS v2 Roadmap

This document outlines the initial roadmap for govOS v2 development.

Sprint 1 — Modern authentication
- Passkeys
- Google OAuth
- Apple Sign-In
- Microsoft Sign-In
- Email Magic Links

Sprint 2 — User Dashboard
- Active Cases
- Saved Documents
- Recent Activity
- Notifications
- AI Assistant

Sprint 3 — Tenant Assistant
- Guided Interview
- Timeline Builder
- Document Generator
- Evidence Vault
- AI Guidance

Sprint 4 — Case Management Engine
- Case model
- Timeline, Documents, Notes, AI context, Status

Sprint 5 — Document Generator
- PDF, DOCX, HTML from templates

Sprint 6 — Modern UI
- Next.js, Tailwind, Motion, Mobile-first, WCAG 2.2 AA

Sprint 7 — AI Integration
- RAG, local/hosted models, knowledge integration
